Kimu is a terminal-based chatbot built in Python that responds to user input using simple rules and API integrations.

Features:-
Live weather updates using OpenWeatherMap API.
Quiz questions from Open Trivia Database.
Basic math operations (add, multiply).
Food-based responses and greetings.
Interactive help/menu system.

Technologies Used:-
Python.
Requests for API calls.
Random & HTML for quiz handling.
External APIs for real-time data.
