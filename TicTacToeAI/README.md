This is a simple Tic-Tac-Toe game made using Python where you play against an AI called Kimu.
The game runs in the terminal and the AI is smart enough to always play the best move.

You play as X, and Kimu plays as O.

Features:-
Human vs AI gameplay.
Intelligent AI using Minimax.
Clean terminal-based interface.
Handles win, loss, and draw conditions.
Beginner-friendly and well-structured code.

How it works:-
The game board is represented as a list of 9 positions.
The AI evaluates all possible moves using the Minimax algorithm.
Scores are assigned to each move to choose the best possible outcome.
The game continues until a win or draw is detected.
