This project is a simple movie recommendation chatbot built using Python. 

What was used:-
Python for core logic and CLI interaction.
RapidFuzz for fuzzy matching to handle spelling mistakes and partial inputs.
JSON for storing movie data.
movies.json as a lightweight local dataset containing movie titles and genres.

How it works:-
User interacts with the chatbot named Kimu.
Input is matched against movie titles or genres using fuzzy matching.
Recommendations are generated by comparing shared genres.
Top matches are displayed based on relevance.
